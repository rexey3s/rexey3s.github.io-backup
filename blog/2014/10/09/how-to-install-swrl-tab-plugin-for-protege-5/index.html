How to Install SWRL Tab Plugin for Protege 5

Protege Team has recently pushed out a new SWRL API (included SWRL API, SQWRL API, Drools Rule Engine and a SWRL Tab Plugin). SWRL Tab Plugin is a powerful tools that can help you manipuladte rule and query OWL by SQWRL. The tool â€˜s come out so many dependencies are missing that you have to build and install from source. Here is the guide.

Prerequisites

  • Protege 5
  • Java 7
  • Maven

Install maven for Ubuntu

SWRL API, SWRL Drools Engine

  1. Build and install SWRL API to local maven repositories

Clone this repo SWRL API

or if you run into a problem of missing dependencies , follow this way

  1. Build and install SWRL Drools Engine to local maven repositories

Clone this repo SWRL Drools Engine

Build and install SWRL Tab plugin

Check out this repo

Then copy the jar file in target folder to the plugin folder in Protege_5 folder
